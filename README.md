## Description
The second project must meet the following criteria:
<ul>
  <li>Create a real-world full-stack application to showcase to potential employers.</li>
  <li>As developers, we must use the following technologies: Node.js, Express.js, RESTful API, Handlebars.js, MySQL, Sequalize ORM.</li>
  <li>Must include one new library, package, or technology that was not previously learned.</li>
  <li>Have a folder structure that meets the MVC paradigm.</li>
  <li>Must include authentication (express-session and cookies).</li>
  <li>Protect API keys and sensitive information with environment variables.</li>
  <li>Have the application be deployed using Heroku.</li>
  <li>Must have a polished UI.</li>
  <li>Must be interactive (i.e., accept and respond to user input).</li>
  <li>Meet good-quality coding standards (file structure, naming conventions, follows best practices for class/id naming conventions, indentation, quality comments, etc.).</li>
  <li>Have a professional README (with unique name, description, technologies used, screenshot, and link to deployed application).</li>
</ul>
